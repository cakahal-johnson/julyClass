Switch||Statement

The switch Statement is used to perform different actions based on different conditions

The switch Statement is used also to select one of many code blocks to be executed

Syntax:

initialize the variable
switch(expression) {
case var:
// code block
break;
case var:
// code block
break;
default:
// code block
}

How to use the Switch Statement

  • The switch expression is evaluated once
  • where the value of the expression is compared with the values of each case
  • Then the codition will check if there is a match, the match needs to associate with the block code to execute
  • Where there is no match, the default code block is executed
