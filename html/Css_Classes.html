    This is used to define a special state of an element

  • Styling A Tag for visited and unvisited links differently
  • Styling an Elements when it gets focus

    Other Pseudo-class

    1. :first-child which matches a specified element that is the first-child of another element
    2. :disabled is used in input:disabled which selects every disabled input-element
    3. :enabled is used in input:enabled which selects every enabled input-element
    4. :checked is used in input:checked which selects every checked input-element
    5. :empty used when a p:empty which selects all < p > elements that has no children
    6. :first-of-type is used in p:first-of-type which selects all P-element that is first P-element of it's parents
    7. :root selects the documents' root element
