Media||Queries

Media Queries is technique for delivering a tailored style sheet of different devices

Properties of Media Queries

  1. width and height of the viewport
  2. width and height of the device
  3. Orientation i.e viewing in landscape or portrait mode

Our Focus: for viewports

  • desktops
  • tablets
  • mobile

    • Media Types

  • all: means that is for all media type devices
  • print: is used for printers type
  • screen: is used for computer screens, tablets, samrt-phones type
  • speech: is used for screenreaders for reading out pages type

    • Units Rule for Media Queries

  • max-width : means that the bigger screen comes first
  • min-width : means that the smaller screen comes first
