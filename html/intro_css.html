Intro to css

Definition: 
        Cascading Style Sheets

It decribes how html Elements are to be displayed on Screen, paper, or in other media

  • Css saves a lot of time while working, it can control the layout of multiple web pages all at a once

    • Types of css

  • Inline Css Styling: which is a way of styling HTML within it's Elements Tags as an attribute
  • Internal CSS Styling: this is placed above the body opening tag below the head closing tag by using style tag
  • External Css Styling: this is a hand as a stylesheets place outside the HTML file then we link tag inside HTML head section to connect

    • Css Syntax

  • it starts with Selector e.g HTML Elements tags, class Selector uses 'dot . as selector sign' as attribute and id uses 'hash # as a selector ' as well, * universal means the whole html body

    Css Declaration: has property and value i.e { each cural block has property:value; }

    Css Comments: same as crlt/ but the sign is /* */
