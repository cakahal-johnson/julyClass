Loops

Loops can execute a block of code a number of times...you can run the same function over and over again but each time with a different values

Kinds of Loops

  1. FOR-LOOP: loops through a block of a code a number of time given
  2. While-loop: loops through a block of code while a specified condition is true
  3. While is same as Do-while loops

    4. Syntax for FOR-LOOP

    for(statement 1; statement 2; statement 3){
    //TODO code here
    }

    statement 1: is executed (only one time) before the execution of the code block

    statement 2: this defines the condition for executing the code block

    statement 3: this also execute (every time) after the code block has been executed

    While-loop

    Here the Loop execute a the block code as long as a specified condition is true

    while(condition){
    // TODO code here!!!
    }

    DO While-loop

    Do-while-loop is a variant of the while-loop, then the loop will have to execute the code once, before checking if the condition is true, then it will repeat the loop as loop the condition is true

    do{
    // TODO code here
    }
    while(condition);
